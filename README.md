# Quiz Telemetry Pipeline

## Overview
This repository contains the Quiz Telemetry Pipeline, designed to effectively process and analyze quiz telemetry data.

## Features
- Data collection from various sources
- Real-time data processing
- Comprehensive analytics dashboard

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/Baruj/quiz-telemetry-pipeline.git
   ```
2. Install the necessary dependencies:
   ```bash
   cd quiz-telemetry-pipeline
   npm install
   ```

## Usage
To start the server, run:
```bash
npm start
```

Visit `http://localhost:3000` for the dashboard.
